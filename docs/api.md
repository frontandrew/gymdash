Получение списка чемпионатов
GET /championships/list?types=&fromYear=&toYear=
Параметры:

types: online - показать только онлайн-чемпионаты, gp - только gymkhanaGP, offline - только оффлайн-чемпионаты. Для выбора нескольких значений перечислите их через запятую. Значение по умолчанию: все. Необязательное значение.
fromYear: начиная с какого года получить чемпионаты. Формат - 4 цифры. Если не указан ни fromYear, ни toYear, будут показаны чемпионаты только за текущий год. Необязательное значение.
toYear: до какого года получить чемпионаты. Формат - 4 цифры. Если не указан ни fromYear, ни toYear, будут показаны чемпионаты только за текущий год. Необязательное значение.
Формат ответа:

{
  "id": "int", // ID чемпионата
  "title": "string", //Название чемпионата
  "year": "string", //Год проведения чемионата
  "description": "html", //Описание чемпионата
  "stagesCount": "int", //Количество этапов
  "type": "string" //Тип чемпионата, одно из значений: online, offline, gp
}

Калькулятор
GET /calculate/get?class=&time=
Параметры:

class: Класс, от которого будет рассчитан результат. В качестве класса можно указать его id или буквенное обозначение на русском или английском языке, регистр не важен.
time: Время, которое показал спортсмен этого класса. Формат времени: mm:ss.msec (минуты:секунды.миллисекунды). Помимо этого, можно указать mm:ss,msec или mm:ss:msec или mm:ss или mm. В остальных случаях вернётся ошибка. Минут и секунд не может быть больше 60, миллисекунды округляются до 3 знаков.
Примеры корректных запросов:

GET /calculate/get?class=д1&time=01:21,54
GET /calculate/get?class=d1&time=01:21:54
GET /calculate/get?class=D1&time=01:21.54
GET /calculate/get?class=d1&time=01:21
Примеры некорректных запросов:
GET /calculate/get?class=д1&time=01,21
GET /calculate/get?class=д1&time=01.21
GET /calculate/get?class=дд&time=01:21
GET /calculate/get?class=д1&time=01:555

{
  "referenceTime": "string", //Эталонное время. Формат минуты:секунды.миллисекунды

      "endTime": "string" //Верхняя граница попадания в класс. Формат минуты:секунды.миллисекунды

  ]
}
